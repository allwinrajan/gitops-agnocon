=== Automated Bootstrap (init container) ===
  The following manual steps are now fully automated on every deploy:

  1. pip install mysql-connector-python PyMySQL
     → MySQL now appears in Settings → Database Connections UI
  2. superset db upgrade
     → All Alembic migrations applied (idempotent)
  3. superset fab create-admin
     → Admin user created (idempotent, safe on re-deploy)
  4. superset init
     → Roles & permissions synced (Admin/Alpha/Gamma/sql_lab)
